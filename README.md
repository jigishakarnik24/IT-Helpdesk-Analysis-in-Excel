# IT Helpdesk Dashboard (Excel Project)

## Overview
The IT Helpdesk Dashboard is an Excel-based analytics project designed to monitor and evaluate IT support performance. It provides a centralized view of ticket management, SLA compliance, agent efficiency, and resolution trends to enable data-driven decision-making.

---

## Objectives
- Analyze ticket volume and distribution
- Monitor SLA compliance and breaches
- Evaluate agent performance and efficiency
- Track average resolution time
- Identify operational bottlenecks and improvement areas

---

## Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for interactivity
- Data Cleaning & Formatting
- Formula-based KPI calculations

---

## Dataset Description
- Total Records: 100 tickets
- Time Period: 6 months
- Key Fields:
  - Ticket ID
  - Date Raised
  - Department
  - Priority (High/Medium/Low)
  - Assigned Agent
  - Resolution Time (hrs)
  - SLA Breached (Yes/No)
  - Status
  - Customer Satisfaction

---

## Dashboard Components

### 1. KPI Metrics
- Total Tickets: 100
- SLA Breach Rate: 51%
- Average Resolution Time: 22.93 hours

---

### 2. Ticket Analysis

#### Priority Distribution
- Majority of tickets fall under Medium priority
- High-priority tickets require faster response and monitoring

#### Department-wise Tickets
- Highest tickets from HR and Finance departments
- Indicates higher support demand in these areas

---

### 3. SLA Compliance Analysis
- Over 50% of tickets breached SLA
- Highlights inefficiencies in resolution timelines
- Suggests need for better resource allocation

---

### 4. Agent Performance Analysis
- Comparison of average resolution time per agent
- Identification of top-performing and underperforming agents
- Helps in workload balancing and performance evaluation

---

### 5. Time Trend Analysis
- Monthly ticket trends over 6 months
- Helps identify peak workload periods
- Useful for forecasting and planning resources

---

## Key Insights
- High SLA breach rate indicates process inefficiency
- Medium priority tickets dominate workload
- Certain departments generate more support requests
- Performance varies across agents, indicating scope for optimization
- Resolution time can be improved with better task allocation

---

## Outcomes
- Created an interactive dashboard for IT performance monitoring
- Identified critical areas needing improvement (SLA, workload distribution)
- Enabled quick and clear visualization of helpdesk operations
- Supported data-driven decision-making for service improvement

---

## Limitations
- Static dataset (no real-time data connection)
- Limited to descriptive analytics (no predictive modeling)
- Uniform SLA assumption across all ticket types

---

## Conclusion
This project demonstrates how Excel can be effectively used to analyze IT helpdesk operations. The dashboard provides actionable insights into ticket handling, SLA performance, and agent efficiency, helping organizations improve service quality and operational efficiency.
